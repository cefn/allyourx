allyourx by cefn

Code examples and community-managed files for the AllyourX standard. Includes Ally, a reference implementation of an XML editor with validation and autocompletion against AllyourX - a dialect of RelaxNG with input-form annotations.

Dependencies

The Ally validating editor currently relies on Firefox as a client-side javascript environment. Tests are already green in a Webkit-based browser. When stable, it will be ported to IE and Opera. To build the Allyourx.org website, you simply need a unix scripting environment and Java.

Install

To run the test suite, simply download the 'ally' directory and visit ally/control/unit/unit.html To incorporate Ally into a project, simply include the relevant javascript files (use the unit test page as a reference) and make invocations on the library to load your schema and data files into the editor.
